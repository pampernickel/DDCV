DDCV
====

A Visualized Shiny App to Evaluate Drug-Drug Interaction.

Project status: **beta**

Author: Tongwu Zhang (xtmgah AT gmail.com)


Dependencies
------------
Tested with:
* R

* ggplot2

* shiny

* 


Installation
------------

```bash
git clone "https://github.com/xtmgah/DDCV.git"
cd DDCV.git
```
